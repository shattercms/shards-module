> ShatterCMS submodule for handling block-based web content.

## Getting Started

### Installation

```bash
# npm
$ npm install @shattercms/shards

# yarn
$ yarn add @shattercms/shards
```

Add the module to your ShatterCMS config file.

```ts
export default {
  modules: ['@shattercms/shards'],
};
```
